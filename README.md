# VALORANT Champions Tour (VCT) Match Predictor

Predict the outcome of official VCT matches using ML.

## Tech Stack

### Frontend
- **React**
- **Vite**

### Backend
- **Flask (Python)**

---

## Features

- Predicts VCT match winners based on team statistics and historical data using a random forest classifier

---

## Setup
```bash
git clone <repository-url>
cd VCT-Outcome-Predictor-Website

# Backend setup
cd server
pip install -r requirements.txt
python app.py 

# Frontend setup in second terminal
cd client
npm install
npm run dev
```
